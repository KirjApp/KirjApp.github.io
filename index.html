KirjApp application


"About Us"

Development of a book review app. A Group project of 4 contributors. Allows users to search for books, create a profile, log-in and write reviews for the books.


Projektin idea on kirja-arvostelusovellus. Sovelluksella käyttäjä voi hakea kirjoja valmiista tietokannasta, kirjoittaa arvostelun kirjasta ja lukea muiden tekemiä arvosteluja kirjoista.

Sovellus ratkaisee ongelman, jossa käyttäjä on lukenut kirjan, muttei muista myöhemmin, minkälainen se oli. Käyttäjä voi kirjoittaa kirjasta arvostelun luettuaan sen ja palata arvosteluun myöhemmin. Sovellus ratkaisee myös ongelman, jossa käyttäjä ei tiedä, minkä kirjan lukisi seuraavaksi. Käyttäjä voi katsoa muiden tekemiä arvosteluja kirjoista.

KirjApp is hosted on Heroku

Technologies used

Detailed description to be filled later as well as shared GitHub repositories. KirjApp is a React App using Material-UI libraries for front end in addition to a component for back end connecting to MongoDB for database functionality. The information is fetched using Google Books API.
