KirjApp application


"About Us"

Development of a book review app. A Group project with 4 contributors. The main purpose of the project was to create a work sample to showcase our abilities.

The app allows users to search for books, create a profile, log in, read and write reviews for the books. The app solves the problem where a user has read a book, but does not remember later what the book was like. The user can write a review for the book, and return to it at a later time. The app also solves the problem where the user does not know which book to read next. The user can search for books and read reviews written by other people.


Projektin idea on kirja-arvostelusovellus. Projekti toteutettiin 4 hengen ryhmässä. Projektin ensisijainen tarkoitus oli työnäytteen teko.

Sovelluksella käyttäjä voi hakea kirjoja valmiista tietokannasta, kirjoittaa arvostelun kirjasta ja lukea muiden tekemiä arvosteluja kirjoista.​ Sovellus ratkaisee ongelman, jossa käyttäjä on lukenut kirjan, muttei muista myöhemmin, minkälainen se oli. Käyttäjä voi kirjoittaa kirjasta arvostelun luettuaan sen ja palata arvosteluun myöhemmin.​ Sovellus ratkaisee myös ongelman, jossa käyttäjä ei tiedä, minkä kirjan lukisi seuraavaksi. Käyttäjä voi etsiä kirjoja ja lukea muiden tekemiä arvosteluja.

KirjApp is hosted on Heroku
KirjApp source code on GitHub


Technologies used

KirjApp is a React App using Material-UI libraries. The information is fetched over Google Books API and presented in the Front End part of the App that works in a web browser. The App is designed to have a Back End functionality connecting to the MongoDB for database functionality.
